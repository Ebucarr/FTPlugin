插件安装说明:
1. 如果你当前的windows 富途牛牛的版本号低于3.26, 请在QQ群共享文件中下载更新
2. pluginserver.dll 是已经编译成功的release版本， 如有需要，可自行编译.
3. 将config.ini, pluginserver.dll  copy 到ftnn安装目录下的plugin子目录 
4. 建议安装pycharm 调试运行, 请在百度中自行搜索下载
5. python 版本 2.7及上支持
6. open_quote.py是富途官方提供的python接口, 保持更新中。
